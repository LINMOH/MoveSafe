## 第一部分：系统概述

### 1.1 项目背景
随着健康意识的提升，越来越多人参与户外运动。但运动环境的安全性常常被忽视。

恶劣的天气、污染的空气、拥挤的场地都可能影响运动效果甚至带来健康风险。

### 1.2 解决方案
MoveSafe系统通过实时数据分析和科学算法，为用户提供：
- 环境安全评估
- 个性化健康建议
- 实时体育资讯
- 运动知识科普

### 1.3 技术亮点
- 前后端分离架构
- 可解释性AI算法
- 智能降级策略
- 响应式设计

## 第三部分：核心功能演示

### 3.1 环境评估功能
让我演示一下环境评估功能：

**操作步骤**：
1. 选择城市获取实时天气
2. 输入环境参数
3. 系统自动计算适宜度分数
4. 查看详细建议

**算法原理**：
系统采用加权评分算法，考虑五个因素：
- 空气质量（权重30%）
- 温度（权重20%）
- 湿度（权重10%）
- 人流密度（权重20%）
- 天气状况（权重20%）

每个因素都有科学的评分标准，最终得出0-100的综合分数。

### 3.2 健康评估功能
健康评估基于用户的生物指标：

**计算指标**：
1. BMI（身体质量指数）
2. BMR（基础代谢率）
3. TDEE（每日总消耗）
4. 最大心率

**个性化建议**：
系统会根据用户的年龄、BMI、季节等因素，生成针对性的运动建议。

比如：BMI超过25且年龄大于45岁的用户，会收到关节保护建议。

## 第五部分：算法原理

### 5.1 环境评估算法
```javascript
// 简化版算法
function calculateScore(params) {
  // 各因素评分
  const airQualityScore = getAirQualityScore(params.aqi)
  const temperatureScore = getTemperatureScore(params.temp)
  const humidityScore = getHumidityScore(params.humidity)
  
  // 加权计算
  const totalScore = 
    airQualityScore * 0.3 +
    temperatureScore * 0.2 +
    humidityScore * 0.1 +
    // ... 其他因素
  
  return Math.max(0, Math.min(100, totalScore))
}
```
